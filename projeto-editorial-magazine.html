O projeto editorial

Este projeto explorou os fundamentos do design editorial aplicados a uma revista de cultura e comportamento. O objetivo principal foi criar um sistema de grid que garantisse flexibilidade criativa sem abrir mão da consistência visual ao longo das 48 páginas.

O desafio editorial mais interessante foi equilibrar seções com densidades diferentes de texto — matérias longas, ensaios fotográficos e infográficos — mantendo um ritmo de leitura agradável.

Sistema tipográfico

A hierarquia tipográfica foi construída com dois typefaces: uma serif clássica para títulos e chamadas, e uma sans-serif para o corpo de texto. A combinação criou personalidade editorial sem sacrificar legibilidade.

  • Grid de 12 colunas com variações para layouts de 2, 3 e 4 colunas
  • Sistema de estilos de parágrafo com 14 variações diferentes
  • Páginas-mestre para 6 tipos de layout recorrentes na publicação
  • Formatação de legendas, notas de rodapé e destaques laterais
  • Paleta de 3 cores de destaque aplicadas em seções temáticas

"Bom design editorial é quase invisível — o leitor flui pelo conteúdo sem perceber o sistema que organiza tudo por trás."

Processo no InDesign

Todo o projeto foi desenvolvido no InDesign com atenção especial ao uso de estilos de objeto e parágrafo, master pages e bibliotecas de elementos. Essa organização garantiu que revisões de conteúdo não comprometessem o layout das páginas adjacentes.
